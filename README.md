# 🌾 PyDew Valley

**PyDew Valley** is a top-down 2D farming simulation game developed as an academic project. It offers an immersive experience where players manage their own farm through various agricultural activities, from soil cultivation to crop trading.

## 🚀 Key Features
* **Full Farming System:** Tool-based interactions for tilling soil, planting seeds, watering, and harvesting mature plants.
* **Resource Economics:** Integrated trading system with a dedicated trader for selling harvests and purchasing new seeds.
* **Dynamic Environment:** Real-time day-night cycles with smooth sky color transitions and weather effects (rain automatically waters crops).
* **Strategic Management:** Resource and time management mechanics designed to maximize farm productivity and profit.

## 🛠️ Technical Implementation
* **Language:** Python 3.x
* **Core Library:** Pygame
* **Map Parsing:** PyTMX (Tile Map support)
* **Architecture:** Modular system design using **Object-Oriented Programming (OOP)** for maintainable and scalable code.
* **Physics & Movement:** Vector-based movement with normalized directions and **delta-time synchronization** for frame-rate independence.

## 🎮 Gameplay Elements
* **Tool-based Interaction:** Specific tools for specific tasks (Hoe, Watering Can, etc.).
* **Crop Lifecycle:** Realistic management of growth stages across various crop types.
* **Environmental Systems:** Seasonal mechanics that directly influence plant growth and farm layout strategy.
